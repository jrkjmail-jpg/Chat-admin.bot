# Деплой на BotHost

## 1. Подключить GitHub

1. Откройте BotHost.
2. Создайте новый бот-проект.
3. Выберите деплой из GitHub.
4. Подключите репозиторий `jrkjmail-jpg/Chat-admin.bot`.
5. В качестве команды запуска укажите:

```bash
python bot.py
```

Если BotHost использует Docker, он возьмёт `Dockerfile` из репозитория.

## 2. Переменные окружения

Добавьте в настройках проекта:

```env
BOT_TOKEN=токен_из_BotFather
OPENAI_API_KEY=ключ_OpenAI
ADMIN_IDS=ваш_telegram_id
DATABASE_PATH=studio_admin.db
OPENAI_MODEL=gpt-4.1-mini
OPENAI_EMBEDDING_MODEL=text-embedding-3-small
BOT_DEFAULT_MODE=outside_working_hours
STUDIO_TIMEZONE=Europe/Moscow
```

`SERVICE_CHAT_ID` можно не указывать сразу. Его можно назначить командой `/set_service_chat` в Telegram.

## 3. Настройка Telegram

1. Создайте бота через BotFather.
2. Отключите Privacy Mode через BotFather, иначе бот не сможет видеть все сообщения в группах.
3. Добавьте бота в сервисный чат администраторов.
4. Напишите в сервисном чате `/set_service_chat`.
5. Добавьте бота в родительский чат.
6. Напишите в родительском чате `/add_parent_chat`.
7. Дайте боту права администратора, если хотите, чтобы он удалял рекламу и грубые сообщения.

## 4. Проверка

В сервисный чат отправьте текст:

```text
Концерт 25 июня. Сбор в 10:30. Адрес: ДК Молодёжный. Форма: чёрные легинсы, студийная футболка, белые носки.
```

Нажмите кнопку `Добавить в базу`.

В родительском чате спросите:

```text
Во сколько сбор на концерт?
```

Бот должен ответить по базе знаний.

## 5. Важное ограничение MVP

В первой версии бот умеет добавлять в базу текст и PDF. Скриншоты и фото будут добавлены следующим этапом через OCR/OpenAI Vision.
