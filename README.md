# Chat Admin Bot

Telegram AI-администратор для родительских чатов студии.

## Что умеет MVP

- Работает в родительских чатах.
- Принимает материалы в сервисном чате.
- Добавляет знания только после подтверждения администратора.
- Отвечает через OpenAI API только на основании сохранённой базы знаний.
- Если ответа нет, пишет: «Передаю ваш вопрос администратору.»
- Поддерживает режимы работы бота по расписанию.
- Делает базовую модерацию: мат, реклама, подозрительные ссылки.

## Переменные окружения

Создайте переменные в BotHost или в `.env` локально:

```env
BOT_TOKEN=telegram_bot_token
OPENAI_API_KEY=openai_api_key
ADMIN_IDS=123456789,987654321
SERVICE_CHAT_ID=-1001234567890
DATABASE_PATH=studio_admin.db
OPENAI_MODEL=gpt-4.1-mini
OPENAI_EMBEDDING_MODEL=text-embedding-3-small
BOT_DEFAULT_MODE=outside_working_hours
STUDIO_TIMEZONE=Europe/Moscow
```

## Режимы работы

- `always` — бот отвечает всегда.
- `outside_working_hours` — бот отвечает только вне рабочего времени студии.
- `working_hours_only` — бот отвечает только в рабочее время.
- `off` — бот не отвечает на вопросы, но модерация и админ-команды остаются.

По умолчанию рабочее время студии:

```text
Пн-Пт: 16:00-22:00
Сб: 10:00-18:00
Вс: выходной
```

## Запуск локально

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python bot.py
```

## Команды администратора

- `/start` — старт.
- `/set_service_chat` — назначить текущий чат сервисным.
- `/add_parent_chat` — назначить текущий чат родительским.
- `/mode always` — включить круглосуточный режим.
- `/mode outside_working_hours` — отвечать вне работы студии.
- `/mode working_hours_only` — отвечать только в рабочее время.
- `/mode off` — выключить ответы.
- `/hours` — показать рабочее время.

## Как добавлять знания

1. Добавьте бота в сервисный чат.
2. Отправьте туда текст или PDF.
3. Бот сделает выжимку и предложит добавить материал.
4. Администратор нажимает «Добавить в базу».
5. После этого бот сможет отвечать родителям по этой информации.
